# Universal Odoo Gateway

A FastAPI-based universal gateway that connects to any Odoo instance without requiring custom addons.

## Features

- ✅ Universal Odoo connectivity via XML-RPC
- ✅ Multi-tenant support with license-based routing
- ✅ JWT-based authentication
- ✅ Generic CRUD operations for any Odoo model
- ✅ Specialized HR and Payroll endpoints
- ✅ Redis caching for sessions
- ✅ Comprehensive logging and error handling
